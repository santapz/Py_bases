- Se puede abrir en VSC para repasar conceptitos vistos
- 

    LISTAS (class list)
- Colecciones o arrays. Se pueden modificar, agregar datos, quitar datos, sumar listas, buscar en la lista, etc.
- - Se construyen con '[]'

   TUPLAS   (class tuple)
- Son listas INMUTABLES. No se pueden modificar. Ej tupla = (1,2) 
- Se construyen con '()'
- Se pueden imprimir, contar, buscar, etc. 

    CONJUNTOS (Class set)
- Los elemenos se agregan de mandera desordenada y no pueden haber duplicados.
- Se imprimen en orden aleatorio y los duplicados(si los hay) no se muestran 
- Se construyen con '{}'
- Las fn q usen apendices no sirven (.pop, .apend, .insert) 

    FROZENSET (Class frozenset)
- Conjuntos inmutables, no se pueden agregar ni quitar elementos

    DICCIONARIOS (Class dict)
- Desordenados. Sin indices (=Conjuntos)
- 2 elementos por posicion. Clave y Valor
- No se puede duplicar la clave.
- Se construyen con '{clave:valor, ..}'

    PILAS
- Se construyen con listas '[]'
- Se agregan elementos al final (apilar) y
- Se sacan del final(el ultimo q entra) se desapila

    COLA
- Se construyen con listas '[]' 
- Estructura de dato fijo, primero en entrar y primero en irse. ejemplo cola de banco.
